## Default Parameters

### Definição
Caso o valor do parâmetro seja `undefined` é utilizado o valor padrão.
~~~javascript
const greeting = (user = 'usuário') => console.log(`Welcome ${user}!`);

greeting(); // Welcome usuário!
~~~
